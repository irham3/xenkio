> **Free Browser-Based Tools. No Uploads, No Limits.**

Free online tools for PDFs, images, and developer utilities. Everything runs locally in your browser. Your files never leave your device. No sign-up, no limits. GDPR compliant by design.

## The Name Behind Xenkio

### XEN | *From Greek "xenos" (ξένος)*

- **Meaning:** Foreign, new, innovative, different
- **In tech context:** Cutting-edge, modern, forward-thinking

### KIO | *From Japanese "器" (utsuwa/ki)*

- **Meaning:** Tool, vessel, container, instrument
- **Modern interpretation:** "Key" in pronunciation

### XENKIO = **"Innovative Tools"** or **"Key to Innovation"**

## Core Value Proposition

- **100% Client-Side Processing**: All processing happens locally in your browser.
- **Privacy First, Zero Uploads**: Your files never leave your device. No server uploads means GDPR compliance by design.
- **No Limits, No Sign-ups**: Free forever, no file size limitations, no strings attached, and no account creation required.
- **Instant & Offline Capable**: Instant results without waiting for uploads/downloads. Once loaded, tools work seamlessly even without an internet connection.

## About the Tools

Xenkio provides a comprehensive suite of web-based utilities designed to streamline common digital tasks, including:

- **PDF Tools**: Compress, merge, split, converter, and protect.
- **Image Tools**: Resize, compress, format conversion, background remover, and EXIF readers.
- **Developer Utilities**: JSON formatter, JWT debugger, Regex tester, hash generators, and more.
- **Text & Data Tools**: Markdown editor, CSV converter, and text manipulation.

*Everything is built around the core philosophy: your data remains yours.*
